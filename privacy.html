Overview

This extension automates deletion of Facebook Messenger conversations on facebook.com/messages when the user requests it.

Data Collection

We do not collect, transmit, or sell personal data. No message content is collected or stored.

Permissions & Use

  • Host (facebook.com/messages/*) — access the Messenger page DOM to click the on-page Delete controls.
  • activeTab — temporary access only when the user starts the action.
  • scripting — runs the small on-page automation script.
  • storage — saves local, non-personal settings (e.g., delay between deletions).
  • notifications (optional) — local status alerts; contains no personal data.

Sharing

No user data is shared with third parties.

Security / Remote Code

All code is packaged with the extension. The extension does not load or execute remote code and does not use eval.
